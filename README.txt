Arquivos incluídos:
- index.html
- manifest.json
- service-worker.js
- pasta icons

Senha inicial da área administrativa:
1234

Passo a passo:
1. Crie um repositório novo no GitHub.
2. Envie index.html, manifest.json, service-worker.js e a pasta icons.
3. Ative o GitHub Pages em Settings > Pages > main > /(root).
4. Abra o link no Safari.
5. Toque em Compartilhar > Adicionar à Tela de Início.
